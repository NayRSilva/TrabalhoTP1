Detailed Description

\ Classe Jogo extende a classe abstrata Produto

Essa classe é uma especialização da classe abstrata Produto, assim como é a classe Console. Dessa forma, ela faz Override do método abstrato getPreco() e serve para representar os jogos para locação.
